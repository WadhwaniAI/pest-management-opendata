# Wadhwani AI Pest Management Open Data

This repository contains data for training pest recognition
models. Such models are used by the Wadhwani Institute for Artificial
Intelligence to help cotton farmers make better pest management
decisions. The repository also contains scripts to format the data for
consumption by popular neural network frameworks.

Please see the
[wiki](https://github.com/WadhwaniAI/pest-management-opendata/wiki)
for more information.
